# ItemEvents

### Enum Values

| Name         | Description                                      |
| ------------ | ------------------------------------------------ |
| `Command`    | Triggered when the `use` command is used         |
| `Pickup`     | Triggered when a player picks up the item.       |
| `Drop`       | Triggered when the item is dropped.              |
| `Use`        | Triggered when the item is used.                 |
| `Noclip`     | Triggered when a player presses their noclip key |
| `Detonation` | Triggered when the item detonates.               |
| `SSSS`       | Server Specific Settings trigger.                |
