# SCP-244

| Property        | Type    | Description                         |
| --------------- | ------- | ----------------------------------- |
| `ActivationDot` | `float` | The activation dot product.         |
| `Health`        | `float` | The health of the item.             |
| `MaxDiameter`   | `float` | The maximum diameter of the effect. |
| `Primed`        | `bool`  | Whether the item is primed.         |

**Example:**

```yaml
CustomItemType: SCPItem
Item: SCP244a
CustomData:
  activation_dot: 0.5
  health: 100.0
  max_diameter: 5.0
  primed: true
```
