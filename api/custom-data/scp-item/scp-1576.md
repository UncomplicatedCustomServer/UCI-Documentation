# SCP-1576

| Property    | Type         | Description                  |
| ----------- | ------------ | ---------------------------- |
| `Effect`    | `EffectType` | The effect to apply.         |
| `Duration`  | `float`      | The duration of the effect.  |
| `Intensity` | `byte`       | The intensity of the effect. |

**Example:**

```yaml
CustomItemType: SCPItem
Item: SCP1576
CustomData:
  effect: Invisible
  duration: 15.0
  intensity: 1
```
