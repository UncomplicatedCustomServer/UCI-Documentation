# SCP-1344

| Property               | Type   | Description                                 |
| ---------------------- | ------ | ------------------------------------------- |
| `Apply1344Effect`      | `bool` | Whether to apply the SCP-1344 effect.       |
| `ApplyBlindnessEffect` | `bool` | Whether to apply the blindness effect.      |
| `AllowUnequip`         | `bool` | Whether to allow the item to be unequipped. |

**Example:**

```yaml
CustomItemType: SCPItem
Item: SCP1344
CustomData:
  apply_1344_effect: true
  apply_blindness_effect: true
  allow_unequip: false
```
