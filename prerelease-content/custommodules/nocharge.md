# NoCharge

## NoCharge

This custom module prevents the 'Charge' functionality of Jailbird items from succeeding. When a player attempts to start or trigger a charge, it is immediately failed.

### Configuration

This module does not have any configurable properties.

| Value | Description                               |
| ----- | ----------------------------------------- |
| N/A   | This module has no configurable settings. |

### Example

```yaml
custom_modules:
  NoCharge: []
```
