# ItemGlow

Makes the CustomItem pickup glow when dropped

| Value     | Description                                     |
| --------- | ----------------------------------------------- |
| GlowColor | The Hexcode of the color that the glow will be. |
| Range     | The range of the glow.                          |
| Intensity | The intensity of the glow.                      |



```yaml
custom_modules:
  ItemGlow:
  - GlowColor: '#00FF00'
    range: 5
    intensity: 2
```
