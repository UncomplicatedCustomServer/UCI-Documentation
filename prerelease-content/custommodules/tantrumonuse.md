# TantrumOnUse

This custom module triggers a Tantrum hazard when a player uses an item. It attempts to spawn the hazard at the player's position, or directly below them if there's ground within a 3-unit radius.

### Configuration

This module has no configurable properties.

| Value  | Description                                            |
| ------ | ------------------------------------------------------ |
| (None) | This module does not have any configurable properties. |

### Example

```yaml
custom_modules:
  TantrumOnUse: []
```
