# VaporizeKills

This custom module, `VaporizeKills`, handles player deaths caused by custom items. When a player is killed by an item that has the "Vaporize" functionality, this module will trigger the `Vaporize` effect on the victim, causing them to be entirely removed.

### Configuration

This module does not have any configurable properties.

| Value | Description |
| ----- | ----------- |

### Example

```yaml
custom_modules:
  VaporizeKills: []
```
